Gerar arquivo para publicar

O arquivo gerado contém todos os dados atuais embutidos. Qualquer pessoa que abrir o link verá o painel completo — sem precisar de login ou conta.

O arquivo publicado é somente leitura — sua equipe vê tudo mas não consegue editar. Quando lançar um novo mês, gere e publique um novo arquivo.
Passo a passo — Netlify Drop
  1. Clique em "Gerar HTML para publicar" acima — vai abrir uma nova aba no navegador
  2. Na nova aba, pressione Ctrl+S (Windows) ou Cmd+S (Mac)
  3. Na janela que abrir, mude o nome para index.html e salve
  4. No seu computador, crie uma pasta nova (ex: "painel-b2c") e mova o index.html para dentro
  5. Acesse netlify.com/drop no navegador
  6. Arraste a pasta inteira para a área cinza da página
  7. Copie o link e envie para sua equipe!
⚠️ Por que arrastar a pasta e não o arquivo?
O Netlify precisa de uma pasta contendo o arquivo index.html. Se arrastar só o arquivo, a navegação entre as abas do painel não funciona.
🔄 Para atualizar depois
  • Gere um novo index.html aqui
  • Coloque em uma pasta
  • No Netlify, vá em Deploys e arraste a nova pasta
  • O mesmo link continua funcionando
